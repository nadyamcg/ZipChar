# ZipChar

A simple Python script to extract filenames in a zip directory into a C array of strings 

## Installation

Clone the repository with the command 
`git clone https://github.com/lunarmeadow/ZipChar.git`
or download ZIP and extract to a folder of your convenience.

Ensure [Python 3](https://www.python.org/) is installed on your system and up-to-date.
ZipChar has no external dependencies and solely uses the standard library.
